Contentize.js Documentation

Contentize.js

A lightweight JavaScript library for injecting dynamic content from a text file into HTML elements with ease.

Introduction

Contentize.js is a JavaScript micro-library designed to let you dynamically replace placeholders in HTML with content defined in a separate text file or within an embedded 'pre' tag in the HTML. Perfect for small projects, prototyping, or creating simple content-driven pages without needing a full backend.

Features

  • Easy-to-use placeholder syntax for content injection
  • Option to load content from a local file or inline within the HTML
  • No dependencies required, just plain JavaScript
  • Fast setup with minimal code

Usage

The setup for Contentize.js is straightforward. Here’s how to use it:

1. Create a Content File

Define your content in a text file (e.g., copy.txt) with each placeholder prefixed by @, followed by the actual content. Here’s an example:

@headline
The Big Launch

@lead
Welcome to the exciting new way to add content!

@punchline
All the benefits, none of the hassle!

2. HTML Structure with Placeholders

In your HTML file, use placeholders (e.g., @headline) in your markup. Here’s a sample:

<h1>@headline</h1>
<p>@lead</p>
<h2>@punchline</h2>

3. Add Contentize.js

Include the Contentize.js script in your HTML and set the data-content attribute to your text file. Example:

<script src="contentize.js" data-content="copy.txt"></script>

4. Using Inline Content

If you prefer to include the content within the HTML file, use a hidden <pre> tag with an ID:

<pre id="content-data">
@headline
The Big Launch

@lead
Welcome to the exciting new way to add content!

@punchline
All the benefits, none of the hassle!
</pre>

Advantages

  • Simple to set up and lightweight for small projects.
  • No back-end dependency, making it great for static sites.
  • Improves content management by keeping the structure and content separate.
  • Perfect for quick prototyping or adding dynamic content to static HTML pages.
